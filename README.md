# Nest Admin
A admin template based on [React](https://reactjs.org/) & [Ant Design](https://ant.design/)

## Installation
```bash
$ npm install
```

## Running the app
```bash
# development (localhost:5173)
$ npm run dev

# build
$ npm run build

# Deploy
$ npm run deploy
```
